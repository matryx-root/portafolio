## Publicacion en GitHub Pages

1. Crear un repositorio publico llamado `portafolio` en la cuenta `matryx-root`.
2. Subir el contenido completo de esta carpeta al repositorio.
3. En GitHub, entrar a `Settings > Pages`.
4. Seleccionar `Deploy from a branch`.
5. Elegir rama `main` y carpeta `/root`.
6. Guardar y esperar a que GitHub publique el sitio.
